CoinMiningCompare API Documentation

CoinMiningCompare provides API access to the cryptocurrency data listed on coinminingcompare.com with JSON responses. All API requests require an API key. To generate an API key, please click the "Generate API key" button. In order for an API key to be approved a valid facebook account is required.

Your API Key: {{key}}

Bitcoin Difficulty and Block Reward

https://www.coinminingcompare.com/api/coins/btc?key=YOUR_API_KEY
Get the latest difficulty and block reward for bitcoin (BTC)

Sample Response:
{"name":"Bitcoin","symbol":"BTC","difficulty":5178671069072,"reward":12.5,"coinToDollar":"8202.08"}
